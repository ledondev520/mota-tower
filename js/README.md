# js/ — 游戏源码

> 若本文件夹结构或内容变化，请更新本文件。

## 目的 / 边界 / 职责

浏览器端全部游戏代码（普通 script 加载，file:// 可玩）；
分层：数据与公式（可被 Node 复用）→ 引擎（规则）→ 表现（渲染/音效/输入）。

## 文件清单

| 名字 | 地位 | 功能 |
|------|------|------|
| `formulas.js` | 最底层·唯一真相 | 战斗伤害预览 / 胜负判定 / 商店定价的纯函数 |
| `data.js` | 内容层·唯一来源 | 怪物表、道具、商店、NPC、15 层地图令牌与解析 |
| `savecode.js` | 存档编解码器 | 存档码（deflate+base64url）与地图差异 diff 的纯函数 |
| `sprites.js` | 表现层资产 | 16×16 字符矩阵像素画 + 调色板换色 + 预渲染缓存 |
| `engine.js` | 规则中枢 | 移动/战斗/门钥匙/拾取/楼梯/楼传/商店/NPC/存档（v2 差异式+导入导出），经事件推送 UI |
| `ui.js` | 表现层总装 | Canvas 地图渲染、HUD、弹窗、键盘 + 虚拟摇杆/方向键触屏输入、存档码界面 |
| `audio.js` | 表现层听觉 | WebAudio 振荡器合成音效（无音频文件） |
| `main.js` | 装配 | DOM 就绪后启动 UI |
