# 魔塔 · Mota — 15 层经典数值魔塔

> 若本项目功能 / 架构 / 目录发生变化，请同步更新本文件。

一款**零依赖、纯静态**的网页魔塔：确定性数值战斗、三色门钥匙经济、
宝石药水成长、商店与 NPC、怪物手册与伤害预览、楼层传送、三槽存档 + 自动存档。
数值可通关性由验证器**机器证明**，不靠手感。

## 立即游玩

- **线上**：<https://ledondev520.github.io/mota-tower/>（GitHub Pages）
- **本地**：直接双击 `index.html`（离线可玩，无需构建）；
  或 `npm run dev` 后访问 <http://localhost:8123>。
- **键盘**：方向键 / WASD 移动（可长按）· M 怪物手册 · J 楼层传送 ·
  K 存档 · L 读档 · T 切换伤害预览 · Esc 关闭弹窗。
- **触屏**：屏幕下方自带**虚拟摇杆**（拖拽连走）与**方向键**（长按连走），
  一键切换；右侧手册/楼传快捷圆钮。标题页「开始/继续/导入」三入口。

## 存档与跨浏览器迁移

- 自动存档：换层、战斗、开门、拾装备、购买、领奖励后即时写入 auto 槽。
- 手动存档：3 个槽位随时存读；标题页「继续冒险」自动读取**最新**一次记录。
- **跨浏览器**：游戏内「导出存档」生成压缩存档码（可复制 / 下载 .txt），
  任意浏览器标题页「导入存档」粘贴或上传文件即可原样继续（离线可用，无需账号）。

## 游戏规则（经典魔塔约定）

- 战斗无随机：勇士先手，轮流互击。
  勇士每击 = `攻击 − 怪物防御`（≤0 无法战胜）；怪物每击 = `max(0, 怪物攻击 − 勇士防御)`；
  总伤害 = `(ceil(怪HP/勇士每击) − 1) × 怪物每击`。
- 特殊能力：**先攻**（怪物多打一轮）、**魔攻**（无视防御）、**连击**（每轮两击）。
- 致死战斗直接拦截（预计伤害 ≥ 当前生命时禁止进入），不会误操作暴毙。
- 资源：黄/蓝/红钥匙与门；红药 +200 / 蓝药 +500 / 圣水 +2000；
  红宝石 +3 攻 / 蓝宝石 +3 防；三档剑盾；金币商店（价格递增）与钥匙商店。
- 目标：登上 15 层，击败魔王，救出公主。

## 开发命令

| 命令 | 作用 |
|------|------|
| `npm test` | 战斗公式单元测试（node --test） |

## 架构（模块与接口）

```
formulas.js (战斗数学·唯一真相)   savecode.js (存档码编解码/地图差异)
    ↑            ↑                     ↑
engine.js ← data.js (全部数值/地图/文案)
    ↑            ↑
  ui.js ← sprites.js (程序化像素画)  + audio.js (WebAudio 合成音效)
    ↑
 main.js (装配启动)
```

- 引擎与 UI 之间是**事件接口**：引擎执行规则并派发事件，UI 只做视听表现与输入转发。
- `tools/balance_check.mjs` 经 vm 加载与游戏**完全相同**的 `formulas.js` + `data.js`，
  保证"验证的"与"玩到的"是同一套数值。
