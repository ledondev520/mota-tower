# tools/ — 质检工具

> 若本文件夹结构或内容变化，请更新本文件。

## 目的 / 边界 / 职责

Node 侧质量保障：加载与游戏完全相同的 `js/formulas.js` + `js/data.js`，
对战斗公式做单元测试、对全塔数值做机器可通关验证。不含任何游戏运行时代码。

## 文件清单

| 名字 | 地位 | 功能 |
|------|------|------|
| `formulas.test.mjs` | 单元测试 | 8 个用例覆盖普通/先攻/魔攻/连击/致死拦截/定价（`npm test`） |
| `balance_check.mjs` | 数值裁判 | 结构校验 + 资源账本 + 贪心通关模拟 + 逐层曲线（`npm run check`，失败退出码 1） |
